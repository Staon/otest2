Detailed Description

Generic interface of a testcase repeater.

The case repeater is responsible for repeated run of a testcase object. For each run new instance of the case is created. So actually the repeater works as a testcase factory. However, there might be a confusion - the case factory object creates instances of this testcase repeater.
